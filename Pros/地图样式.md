| elementType | 控制内容 | 常用 stylers |
|---|---|---|
| `all` | 几何 + 标注全部 | `visibility` |
| `geometry` | 面/线的形状本体 | `visibility`, `color`, `weight` |
| `geometry.fill` | 面的填充颜色 | `color` |
| `geometry.stroke` | 线/面描边颜色 | `color` |
| `geometry.topfill` | 建筑顶面颜色（3D） | `color` |
| `geometry.sidefill` | 建筑侧面颜色（3D） | `color` |
| `labels` | 所有文字和图标 | `visibility` |
| `labels.text` | 文字整体 | `visibility` |
| `labels.text.fill` | 文字颜色 | `color` |
| `labels.text.stroke` | 文字描边（光晕） | `color` |
| `labels.icon` | 小图标（如 POI 图标） | `visibility` |

## featureType 完整分类列表

### 底图 / 自然地物

| featureType | 含义 |
|---|---|
| `land` | 陆地底色 |
| `water` | 水域（河流、湖泊、海洋） |
| `green` | 绿地 / 植被 |
| `manmade` | 人造物（围栏、广场等） |
| `shopping` | 商业区域 |

### 建筑 / 功能区

| featureType | 含义 |
|---|---|
| `building` | 建筑物 |
| `subwaystation` | 地铁站 |
| `education` | 教育机构区域 |
| `medical` | 医疗机构区域 |
| `scenicspots` | 景区区域 |

### 道路

| featureType | 含义 |
|---|---|
| `road` | 通用道路（快捷，覆盖所有道路） |
| `highway` | 高速公路 |
| `nationalway` | 国道 |
| `provincialway` | 省道 |
| `cityhighway` | 城市高速 |
| `arterial` | 城市主干道 |
| `local` | 地方道路 / 小路 |
| `railway` | 铁路 |
| `subway` | 地铁线路 |

### 道路标牌

| featureType | 含义 |
|---|---|
| `highwaysign` | 高速路牌图标 |
| `nationalwaysign` | 国道路牌图标 |
| `provincialwaysign` | 省道路牌图标 |
| `tertiarywaysign` | 三级路牌图标 |

### 行政区划（几何边界）

| featureType | 含义 |
|---|---|
| `district` | 区/县 几何边界线 |
| `boundary` | 国界 / 行政边界线 |

### 标注文字（地名 / xxxlabel）

| featureType | 含义 | 备注 |
|---|---|---|
| `continent` | 大洲名称 | |
| `country` | 国家名称 | |
| `city` | 城市名称 | |
| `town` | 乡镇名称 | |
| `village` | 村庄名称 | |
| `districtlabel` | 行政区划标注 | ⚠️ 不是 `district` |
| `poilabel` | POI 兴趣点标注 | ⚠️ 不是 `poi` |
| `scenicspotslabel` | 景点标注 | |
| `transportationlabel` | 交通枢纽标注 | |
| `airportlabel` | 机场标注 | |
| `subwaylabel` | 地铁站标注 | |

> ⚠️ **命名规律**：凡是标注文字类的元素，都在类型名后加 `label`（如 `poilabel`、`districtlabel`）。  
> 直接用 `district`、`poi` 只能控制几何边界/区域，无法控制文字标注。

## stylers 可用属性

| 属性 | 值 | 说明 |
|---|---|---|
| `visibility` | `"on"` / `"off"` | 显示/隐藏 |
| `color` | `"#RRGGBBAAff"` | 8位十六进制，最后两位为透明度（`ff`=不透明） |
| `weight` | 数字 | 线宽（道路专用） |

---

## 常用示例

```js
map.setMapStyleV2({
  styleJson: [
    // 隐藏 POI 标注
    { "featureType": "poilabel", "elementType": "all", "stylers": { "visibility": "off" } },
    // 隐藏行政区划标注
    { "featureType": "districtlabel", "elementType": "all", "stylers": { "visibility": "off" } },
    // 隐藏道路
    { "featureType": "road", "elementType": "all", "stylers": { "visibility": "off" } },
    // 隐藏建筑
    { "featureType": "building", "elementType": "all", "stylers": { "visibility": "off" } },
    // 修改陆地底色
    { "featureType": "land", "elementType": "geometry", "stylers": { "color": "#091220ff" } },
    // 只改文字颜色，保留图标
    { "featureType": "highway", "elementType": "labels.text.fill", "stylers": { "color": "#ff0000ff" } }
  ]
});
