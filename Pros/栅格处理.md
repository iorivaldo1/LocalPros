发布百度地图影像服务流程:
1. 获取源数据:
使用QGIS另存为时,选择渲染图像,坐标系选择4326,创建选项选择jpeg压缩
2. 使用gdal裁切影像:
gdalwarp -cutline fw.shp -crop_to_cutline -dstalpha test.tif test_cut.tif
3. 由于dal_retile.py要使用python环境下的proj,我的系统环境下的proj的版本不对应，所以先设置环境
set PROJ_LIB=C:\Python314\Lib\site-packages\osgeo\data\proj
set PROJ_DATA=C:\Python314\Lib\site-packages\osgeo\data\proj
set GDAL_DATA=C:\Python314\Lib\site-packages\osgeo\data\gdal
4. 处理为imageSomaic格式，只切片，不加levels(geoserver默认不支持levels)
gdal_retile.py -v -ps 2048 2048 -co "TILED=YES" -co "COMPRESS=LZW" -targetDir imageDir image.tif
5. 为imageSomaic下的tif生成(内部)金字塔:
Get-ChildItem -Filter *.tif | ForEach-Object { gdaladdo -r average $_.FullName 2 4 8 16 32 64 }//内部
6. 在geoserver中添加imageSomaic，并发布
