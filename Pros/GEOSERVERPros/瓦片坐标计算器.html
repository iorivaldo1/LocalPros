计算步骤详解

1. 将经纬度转换为Web墨卡托坐标

Web墨卡托投影（EPSG:3857）是将地球表面投影到平面上的方法。

mercX = lon * 20037508.34 / 180

mercY = log(tan((90 + lat) * π / 360)) * (20037508.34 / 180)

计算结果: -

2. 计算初始参数

地球周长 = 2 * π * 6378137 ≈ 40075016.68 meters

初始分辨率 = 地球周长 / 瓦片大小(256)

原点偏移 = 地球周长 / 2

当前分辨率 = 初始分辨率 / 2zoom

计算结果: -

3. 计算全局像素坐标

像素坐标原点在左上角，X向右增加，Y向下增加

pixelX = (mercX + originShift) / resolution

pixelY = (originShift - mercY) / resolution

计算结果: -

4. 计算瓦片坐标

瓦片坐标基于像素坐标除以瓦片大小(256)

tileX = floor(pixelX / tileSize)

tileY = floor(pixelY / tileSize)

计算结果: -

5. 计算瓦片内的像素坐标

i = floor(pixelX % tileSize) [X方向正常]

j = floor(tileSize - (pixelY % tileSize) - 1) [Y方向翻转]
